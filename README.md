# Bildordbok

Visual picture dictionary with ARASAAC pictograms for language development.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install bildordbok
```

### Fedora/RPM
```bash
sudo dnf install bildordbok
```
